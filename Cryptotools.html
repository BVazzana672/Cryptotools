Warm up tasks

  • Given a string of text, return a list of numbers where a's become 1's (both upper and lower case), b's are 2's, and so on. Other characters are ignored
  • Given a list of numbers (in the range 1-26) return a string of characters replacing 1's with a's, 2's with b's, and so on.
  • Given a list of numbers, peform an identical computation to each number in the list. For example, add 3 to the number and reduce it mod 26

    Number Theory Computations

  • Given numbers a and b, solve the equation ax+by=gcd(a,b).
  • Given large numbers a, k, n, compute ak (mod n).
  • Given a number n, return its prime factorization.
  • Given a number n, generate a sequence of random numbers of length n

    Ciphers

  • Casear shift (shift each letter by a fixed amount)
  • Affine Cipher (shift and multiply)
  • Monoalphabetic Substitution Cipher
  • Keyword Cipher (special case of above)
  • Vigener Cipher (The Unbreakable Cipher)
  • Pohlig-Hellman (Instead of add or multiplying, we raise to powers)
  • RSA (Works like P-H but you can encrypt without knowing how do decrypt)
  • Playfair Cipher (maybe)
  • Hill Cipher (maybe)

    Cryptanalysis

  • Frequency counts
  • Identifying most frequent digraphs, trigraphs, quadgraphs
  • Monoalphabetic Cryptanalysis - Partially decrypt a ciphertext given some guesses of part of the key.
  • Given n and a string of text, output n strings where the first contains every nth letter starting with the first, the second contains every nth letter starting with the second, and so on.
  • Casear analysis - given a Caesar shifted ciphertext, create a relative frequeny bar graph and display it next to the expected relativey frequency graph, allowing the user to shift the cipher graph.
